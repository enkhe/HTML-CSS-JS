Comprehensive CSS Flexbox Guide

This interactive guide demonstrates the power and versatility of CSS Flexbox for modern web layouts.

1. Display

The display property is fundamental. Setting it to flex or inline-flex on a container enables Flexbox behavior for its direct children (flex items).

display: flex;

Creates a block-level flex container.

Note:

  • flex makes the container a block-level element.
  • inline-flex makes it an inline-level element.

2. Flex Direction

flex-direction establishes the main axis, defining the direction items are placed in the flex container.

flex-direction: row (default)

Items are placed horizontally (left to right in LTR languages).
